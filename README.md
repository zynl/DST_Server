script-for-set-up-DST-linux-server
==================================
A script to set up don't starve together linux server

###Install
debian 7

    wget https://github.com/zynl/script-for-set-up-DST-linux-server/blob/master/install.sh
    chmod +x install.sh
    ./install.sh

###Server Tokens
Plese visit [wikia][1]

###Modify setting.ini
